# STLComplianceSite Feature Set

## Product Definition

STLComplianceSite is the public, static front door for the STL Compliance / Arr suite.

## Owns

- homepage
- product marketing pages
- suite overview
- trust/security messaging
- demo/contact forms
- public education
- comparison content
- SEO metadata
- public maturity messaging

## Does Not Own

- auth
- tenant data
- product workflows
- admin behavior
- operational records
- product APIs

## Core Features

- homepage
- products hub
- NexArr page
- StaffArr page
- TrainArr page
- MaintainArr page
- RoutArr page
- SupplyArr page
- Compliance Core page
- security page
- data ownership page
- demo/contact path
- resources
- privacy/terms
- 404

## Required API Surfaces

- `static site only`
- `client login routes to NexArr or suite entry`

## Completion Definition

A visitor can understand the suite, product ownership, trust posture, and how to request a walkthrough or sign in through the right platform path.
