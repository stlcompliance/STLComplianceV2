## Purpose

AssurArr owns the quality assurance and risk-control workflow layer for STL Compliance / ARR:

- Nonconformance
- Quality holds
- Containment
- Disposition
- Corrective actions
- Preventive actions
- CAPA effectiveness
- Quality audits
- Audit findings
- Supplier quality issues
- Customer complaint quality workflows
- Quality release decisions
- Quality score/status snapshots

AssurArr does not own inventory truth, work order execution truth, order lifecycle truth, supplier master truth, customer master truth, document file truth, personnel master truth, regulatory rulepack meaning, reporting read models, or accounting execution.
