# How to review governing bodies

## Audience
Compliance admins and legal/compliance reviewers

## Product
Compliance Core

## Support Status
Supported by current UI/API

## Purpose
Review the organizations or authorities that issue rules used by Compliance Core.

## Before You Start
- Compliance Core owns governing body catalogs.
- Use counsel or authorized compliance review for legal interpretation where needed.

## Steps
1. Open Compliance Core.
2. Open Registry.
3. Open Governing bodies.
4. Search or filter for the authority you need.
5. Open the governing body detail.
6. Review label, jurisdiction relationship, source, status, and related citations or rulepacks.
7. Edit only if you have authority and the UI exposes an edit workflow.
8. Review related citations or requirements after any change.

## What Happens Next
Governing body data supports citations, rulepacks, applicability, and reports.

## Troubleshooting
- If a governing body is duplicated, resolve it before mapping more citations.
- If the governing body should be imported, use Imports and staged review instead of manual duplicate entry.
