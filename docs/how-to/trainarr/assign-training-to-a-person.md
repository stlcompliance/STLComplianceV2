# How to assign training to a person

## Audience
Training admins, managers, and supervisors

## Product
TrainArr

## Support Status
Intended workflow partially supported by current routes/docs

## Purpose
Create a training assignment for a StaffArr person.

## Before You Start
- The person must exist in StaffArr.
- The training program or qualification should exist in TrainArr.
- The current app exposes Assignments, Manual, Queue, and Evaluation routes; exact manual assignment fields should be confirmed in the UI.

## Steps
1. Open TrainArr.
2. Open Assignments.
3. Use Manual for a direct assignment or Queue for pending assignment candidates.
4. Select the StaffArr person.
5. Select the training program, qualification, or remediation reason.
6. Set due date, priority, trainer, evaluator, and evidence requirements when those fields are available.
7. Create the assignment.
8. Notify the trainee through the configured product or field workflow.

## What Happens Next
TrainArr owns the assignment lifecycle. StaffArr can show readiness and qualification results after TrainArr publishes them.

## Troubleshooting
- If the person is missing, create the person in StaffArr.
- If the program is missing, create or configure the TrainArr program before assigning.
