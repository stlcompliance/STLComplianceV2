# How to create a training program

## Audience
Training admins and program owners

## Product
TrainArr

## Support Status
Intended workflow partially supported by current routes/docs

## Purpose
Define a training program or module that can later be assigned to people.

## Before You Start
- TrainArr owns training programs and modules.
- StaffArr owns the person and job context that may determine who needs training.
- The current app exposes Programs and Create routes, but final create field labels should be confirmed in the UI.

## Steps
1. Open TrainArr.
2. Open Programs.
3. Choose Create if the create view is available.
4. Enter the program name, qualification or job relevance, steps, evidence expectations, and renewal behavior requested by the page.
5. Add evaluation or signoff requirements when the program requires them.
6. Save the program as draft if review is needed.
7. Activate or publish only when the program is ready to assign.

## What Happens Next
The program becomes the TrainArr-owned definition used by assignments, qualification rules, and completion records.

## Troubleshooting
- If the create controls are not present in the tenant, document the program requirements outside the product and create the assignment only after the program builder is enabled.
- If a person or position is missing, fix that in StaffArr.
