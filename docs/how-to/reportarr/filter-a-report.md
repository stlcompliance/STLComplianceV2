# How to filter a report

## Audience
Report users, managers, and auditors

## Product
ReportArr

## Support Status
Supported by current UI/API

## Purpose
Narrow a report or dashboard to the tenant, date range, product, status, location, or other fields available in that report.

## Before You Start
- You need access to the report or dashboard.
- Filter options depend on the report definition and dataset.

## Steps
1. Open ReportArr.
2. Open Reports or Dashboards.
3. Open the report or dashboard.
4. Review available filters and parameters.
5. Set the date range, product, status, location, owner, or other filters shown by the page.
6. Run, refresh, or apply the filters.
7. Review filtered totals, rows, charts, or records.
8. Save a view or update the report definition only if you have builder access and the UI provides that action.

## What Happens Next
Filters change the report view or generated output. They do not change source product records.

## Troubleshooting
- If a filter value is missing, check whether the source product publishes that field to ReportArr.
- If the result looks stale, check freshness or sync status before using the report operationally.
