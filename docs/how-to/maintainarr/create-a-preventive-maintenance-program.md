# How to create a preventive maintenance program

## Audience
Maintenance planners and maintenance admins

## Product
MaintainArr

## Support Status
Supported by current UI/API

## Purpose
Define recurring maintenance for assets based on time, meter, condition, or other planned criteria.

## Before You Start
- MaintainArr owns preventive maintenance programs.
- Assets must exist before they can be enrolled.

## Steps
1. Open MaintainArr.
2. Open PM programs.
3. Choose Create.
4. Enter the program name, asset scope, maintenance tasks, due rules, and required evidence.
5. Add parts, labor, or inspection expectations when known.
6. Save the program.
7. Review the PM program detail for enrolled assets and next due work.
8. Monitor work order generation or due queues from PM and Work orders.

## What Happens Next
MaintainArr uses the PM program to drive planned maintenance and work order demand.

## Troubleshooting
- If expected work orders do not appear, verify the asset is enrolled and the due rules are active.
- If parts are needed, confirm the parts demand path to LoadArr/SupplyArr is configured.
