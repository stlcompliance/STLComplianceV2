# How to create an inspection template

## Audience
Maintenance admins and compliance managers

## Product
MaintainArr

## Support Status
Supported by current UI/API

## Purpose
Define reusable inspection questions or checklist structure for assets.

## Before You Start
- MaintainArr owns maintenance inspection templates.
- Compliance Core may define evidence expectations, but MaintainArr owns execution.

## Steps
1. Open MaintainArr.
2. Open Templates or Inspection templates.
3. Choose Create.
4. Enter the template name, asset applicability, sections, questions, answer types, and pass/fail expectations requested by the page.
5. Include evidence expectations when the template requires photos, documents, or notes.
6. Save the template.
7. Use the template in inspections or preventive maintenance programs.

## What Happens Next
New inspections can reference the template, and failed results can drive defects or work orders.

## Troubleshooting
- If a compliance requirement is missing, confirm the evidence requirement in Compliance Core and then update the MaintainArr template.
- If the template should apply only to certain assets, verify asset type or scope before activation.
