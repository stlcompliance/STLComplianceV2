# How to create an organization unit

## Audience
StaffArr admins and operations leaders

## Product
StaffArr

## Support Status
Supported by current UI/API

## Purpose
Add a department, division, or other internal organization unit that people and permissions can reference.

## Before You Start
- StaffArr owns internal organization structure.
- You need StaffArr organization administration access.

## Steps
1. Open StaffArr.
2. Open Organization Structure.
3. Choose the organization-unit area.
4. Create the new unit using the page create panel.
5. Enter the name, parent unit, status, and any required operating context.
6. Save the unit.
7. Assign people, positions, teams, or reporting relationships to the new unit as needed.

## What Happens Next
The new organization unit becomes the canonical internal structure reference for StaffArr and consuming products.

## Troubleshooting
- If another product needs this unit, reference the StaffArr unit instead of recreating it.
- If the exact create label is different in the current UI, use the organization-unit create panel on Organization Structure.
