## Notes

- When a candidate is marked `unclear_needs_review`, the default action is to inventory the existing product tables/forms before centralizing anything.
- Product-owned operational data, evidence files, and regulatory interpretation remain in their owning products even when they reference a shared canonical entity.
- This inventory is intentionally conservative: shared identity moves first, commercial and execution data stays local.
