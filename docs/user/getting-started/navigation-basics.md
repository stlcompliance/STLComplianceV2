# Navigation Basics

## Who It Is For
New users learning how to move around the suite.

## Before You Start
- You must be signed in.
- You need access to at least one product workspace.

## Steps
1. Use the suite shell to confirm the current tenant and current product.
2. Use the product switcher to move between products.
3. Use the product sidebar or mobile menu to open product pages.
4. Use page headers to find the current page title, primary action, and status where shown.
5. Use detail pages or drawers to review a selected record.
6. Use the user menu when you need to sign out.

## Common Problems
- If a navigation item is missing, your role may not allow it or the product may not expose that page for your account.
- If a record opens read-only, you may have read access without manage access.
- If a product opens but data is missing, check filters, selected tenant, and product permissions.
