# RecordArr User Guide

## What This Product Is For
RecordArr is for document storage, record metadata, file versions, controlled documents, retention schedules, legal holds, document approvals, acknowledgements, evidence file storage, and document access history.

## Who Uses It
- records managers
- compliance users
- auditors
- admins who manage controlled documents
