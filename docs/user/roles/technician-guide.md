# Technician Guide

## What This Role Does
A technician performs assigned maintenance work, reports defects, completes inspections, records labor and evidence, and may use Field Companion for mobile tasks.

## What This Role Can Usually Access
- MaintainArr work orders and defects they created or are assigned to.
- Inspection runs they start.
- Field Companion Field inbox tasks when product entitlements and task assignments exist.

## What This Role Usually Cannot Access
- Usually cannot see all tenant work orders unless manager permissions are granted.
- Usually cannot manage assets, PM programs, inspection templates, or close any work order.
- Cannot move warehouse inventory unless LoadArr access is separately granted.

## Common Daily Tasks
- Open assigned work.
- Add work order tasks, labor, and evidence where allowed.
- Report a defect.
- Complete inspection answers and select **Complete inspection** in Field Companion when all required answers are present.
- Use **Open in MaintainArr** from Field Companion for product details.

## Records This Role Works With
- assigned work order
- defect
- inspection run
- evidence upload
- labor entry

## Notifications This Role May Receive
- Assigned work order tasks
- inspection tasks
- blocked task warnings
- sync failures from Field Companion

## Common Issues
- Task is blocked because qualification or readiness checks failed.
- Inspection cannot be completed until required checklist items are answered.
- Evidence upload fails due to connection or file issue.
