# Vendor User Guide

## What This Role Does
A vendor user responds through vendor-facing SupplyArr portal surfaces when invited, such as quote or vendor order workflows.

## What This Role Can Usually Access
- SupplyArr Vendor portal or vendor order portal links when provided.
- Quote draft and submit actions shown in the portal.
- Vendor order status surfaces where the token or invitation allows it.

## What This Role Usually Cannot Access
- Does not receive general suite access unless separately created as a platform user.
- Does not manage SupplyArr vendor master data from the portal.
- Does not see other vendors, internal inventory, purchasing approvals, or financial execution.

## Common Daily Tasks
- Open the vendor portal link provided by the STL Compliance tenant.
- Select **Create quote draft** or **Refresh quote draft** when available.
- Enter prices and lead times.
- Select **Save line** for quote lines.
- Select **Submit quote** when the quote is ready.

## Records This Role Works With
- quote draft
- quote line
- vendor order status
- vendor document where upload is enabled

## Notifications This Role May Receive
- Quote invitations
- vendor order status requests
- document upload requests where enabled

## Common Issues
- Portal link is expired or invalid.
- Quote cannot be submitted until required lines are saved.
- Vendor order status updates may be unavailable unless the invitation supports them.
