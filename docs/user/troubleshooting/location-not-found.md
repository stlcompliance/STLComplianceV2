# I cannot find a location

## Symptoms
- Location picker is empty.
- Warehouse area, dock, or site is missing.
- Transfer or putaway cannot select a destination.

## Likely Causes
- Location was not created in StaffArr.
- Location is inactive or archived.
- Product setup has not refreshed StaffArr locations.
- Location rules block the movement.

## What to Check
1. Open StaffArr Locations.
2. Confirm site and location status.
3. Check parent site.
4. Review LoadArr location rules when inventory is involved.

## How to Fix
- Create or activate the StaffArr location.
- Correct parent path.
- Refresh product location references.
- Choose an allowed destination.

## Who Can Help
StaffArr admin or LoadArr admin.
