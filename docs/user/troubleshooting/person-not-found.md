# I cannot find a person

## Symptoms
- Person does not appear in picker or People list.
- Assignment cannot find the person.
- Field task does not match expected person.

## Likely Causes
- Person was not created in StaffArr.
- Person is inactive or archived.
- Filters are hiding the person.
- Product reference sync has not refreshed.

## What to Check
1. Search StaffArr People.
2. Check active and inactive filters.
3. Confirm tenant.
4. Check person placement and status.

## How to Fix
- Create or reactivate the StaffArr person if appropriate.
- Correct filters.
- Wait for product reference sync or ask an admin to refresh it.
- Do not create duplicate person records in another product.

## Who Can Help
StaffArr admin.
