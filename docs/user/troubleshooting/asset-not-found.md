# I cannot find an asset

## Symptoms
- Asset picker is missing the asset.
- Work order or defect cannot be tied to the asset.
- Readiness view has no asset.

## Likely Causes
- Asset is not in MaintainArr.
- Asset is inactive or hidden by filters.
- Asset creation UI is not fully available.
- Wrong tenant or product.

## What to Check
1. Open MaintainArr Asset registry.
2. Search by asset tag or name.
3. Check filters and status.
4. Ask a manager whether asset setup is complete.

## How to Fix
- Use the existing asset if found.
- Ask a MaintainArr admin to create or correct the asset.
- Do not use free-text asset names as a substitute for asset master records.

## Who Can Help
MaintainArr manager or admin.
