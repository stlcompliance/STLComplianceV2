# How to filter a report

## Audience
ReportArr users reviewing reports or dashboards.

## Purpose
Narrow report data to the relevant date range, product, status, or source when filters exist.

## Before You Start
- ReportArr access.
- A report, dashboard, or dataset view with filters.

## Steps
1. Open ReportArr.
2. Open **Reports**, **Dashboards**, or the relevant view.
3. Select the report or dashboard.
4. Find the filter controls.
5. Choose date range, product, status, source, or other available filters.
6. Apply or rerun the report if required.
7. Review source freshness and filtered results.

## What Happens Next
ReportArr updates the displayed report or creates a report run using the chosen filters.

## Troubleshooting
- If a filter is missing, the report definition may not expose it.
- If results look wrong, check source product data.
- If a correction is needed, update the owning product, not ReportArr.
