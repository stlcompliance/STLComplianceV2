# How to export a report

## Audience
ReportArr users allowed to export reports.

## Purpose
Create a downloadable report export.

## Before You Start
- ReportArr access.
- Export access.
- A report run, dashboard, dataset, or audit package source.
- Allowed export format such as pdf, csv, xlsx, json, html, or zip when available.

## Steps
1. Open ReportArr.
2. Open **Reports** > **Exports** or the report run export area.
3. Choose the source report, dashboard, table, dataset, audit package, chart, or custom source where offered.
4. Choose the export format.
5. Create the export.
6. Wait for the export job to complete.
7. Download or open the export according to the available action.

## What Happens Next
ReportArr creates the export job. If the export must be retained, store it through RecordArr according to your records process.

## Troubleshooting
- If export is unavailable, check permission and report status.
- If export is empty, check filters and source data.
- If a sensitive export is needed for audit, review access and retention requirements.
