# How to prepare financial handoff packets

## Audience
Order coordinators, billing preparation users, and managers.

## Purpose
Prepare invoice-ready and bill-ready packets from completed order context without making OrdArr the accounting system.

## Before You Start
- OrdArr access.
- Permission to prepare or export packets.
- Closed or completion-ready order request.
- Required completion evidence and execution summaries.

## Steps
1. Open OrdArr.
2. Open the order request.
3. Open Completion packets, Invoice-ready packets, or Bill-ready packets.
4. Review customer context, order lines, completion evidence, execution summaries, and packet warnings.
5. Resolve missing or conflicting packet data.
6. Prepare or refresh the packet.
7. Export or hand off the packet only through the approved finance integration or manual finance process.

## What Happens Next
OrdArr owns packet preparation and packet status. External finance systems own invoices, bills, payments, tax, and general ledger posting.

## Troubleshooting
- If packet totals or references are incomplete, review order lines and execution summaries.
- If evidence is missing, check RecordArr and the execution product.
- If accounting status is needed, check the finance system rather than editing OrdArr.
