# How to check in a dock appointment

## Audience
Warehouse receivers and dock staff.

## Purpose
Review dock schedule work and begin receiving or dock handling.

## Before You Start
- LoadArr access.
- Dock schedule read or manage access.
- A scheduled dock appointment or task.

## Steps
1. Open LoadArr.
2. Open **Work** > **Dock Schedule**.
3. Find the appointment or dock task.
4. Review carrier, time, dock, and expected receipt details where shown.
5. Open the related receiving session or expected receipt.
6. Start receiving or update the dock task using the available action.
7. Record exceptions if the arrival is late, missing, or mismatched.

## What Happens Next
The dock task remains tied to receiving and warehouse workflow. Receiving completion happens in LoadArr Receiving.

## Troubleshooting
- If the appointment is missing, check date filters and expected receipt setup.
- If the dock or location is missing, check StaffArr location setup and LoadArr setup.
- If the truck is late, record an exception or note if the workflow supports it.
