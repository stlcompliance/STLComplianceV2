# How to release items from quarantine

## Audience
Warehouse supervisors and authorized inventory users.

## Purpose
Release held or quarantined stock back to availability after review.

## Before You Start
- LoadArr access.
- Inventory release permission.
- Hold or quarantine record.
- Approval or review result if required.

## Steps
1. Open LoadArr.
2. Open **Work** > **Holds** or **Work** > **Exceptions** > **Quarantine**.
3. Select the hold or quarantined item.
4. Review reason, item, quantity, and evidence.
5. Confirm release is allowed by quality or compliance review.
6. Select the release action shown by the page.
7. Enter notes if required.
8. Save the release.

## What Happens Next
LoadArr updates hold status and inventory availability. The release history remains available for audit and stock ledger review.

## Troubleshooting
- If release is disabled, check permission or unresolved review requirements.
- If the stock remains unavailable, check other holds or reservations.
- If evidence is missing, attach or reference the required review record before release.
