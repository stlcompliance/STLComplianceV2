# How to put away inventory

## Audience
Warehouse receivers and inventory staff.

## Purpose
Complete putaway from receiving or staging into the final storage location.

## Before You Start
- LoadArr access.
- Putaway task.
- Destination location.
- Received or staged inventory.

## Steps
1. Open LoadArr.
2. Open **Work** > **Putaway**.
3. Select the putaway task.
4. Review item, quantity, source, and suggested destination.
5. Confirm or choose the destination location.
6. Physically move the stock.
7. Select the putaway completion action shown by the task.
8. Review inventory balance or movement history.

## What Happens Next
LoadArr records the putaway movement and updates stock availability according to holds or restrictions.

## Troubleshooting
- If the task is missing, confirm receiving created a putaway task.
- If the destination is invalid, check location rules.
- If the item is on hold or quarantined, release it before normal putaway if policy requires.
