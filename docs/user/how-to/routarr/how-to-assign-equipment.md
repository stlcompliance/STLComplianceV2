# How to assign equipment

## Audience
Dispatchers and RoutArr managers.

## Purpose
Assign vehicle or equipment context to a RoutArr trip.

## Before You Start
- RoutArr access.
- Equipment assignment access.
- Trip record.
- Vehicle or equipment reference.
- Maintenance readiness when required.

## Steps
1. Open RoutArr.
2. Open **Trips** or **Dispatch board**.
3. Select the trip.
4. Open the equipment or vehicle assignment area.
5. Choose the equipment.
6. Review readiness or validation blockers.
7. Save the assignment.
8. Resolve maintenance blockers in MaintainArr if shown.

## What Happens Next
RoutArr records the equipment assignment. MaintainArr remains the source of truth for vehicle maintenance readiness.

## Troubleshooting
- If equipment is missing, check the equipment reference setup.
- If assignment is blocked, review MaintainArr asset readiness.
- If assignment action is missing, check dispatcher or manager access.
