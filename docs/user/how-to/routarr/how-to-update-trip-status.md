# How to update trip status

## Audience
Dispatchers and drivers with trip execution access.

## Purpose
Move a trip through execution states using RoutArr.

## Before You Start
- RoutArr access.
- Trip execution access.
- Trip record assigned or visible to you.

## Steps
1. Open RoutArr.
2. Open **Trips**, **Dispatch board**, or **Driver portal**.
3. Select the trip.
4. Review validation blockers before changing status.
5. Use the available trip status or execution action.
6. Enter required notes, proof, or timing where shown.
7. Save the update.
8. Review the trip timeline or proof review queue.

## What Happens Next
RoutArr records the trip status update and any related proof or exception records.

## Troubleshooting
- If status cannot move forward, check driver assignment, equipment assignment, and blockers.
- Drivers may only access trips they created or are assigned to drive.
- If proof is required, submit proof before completion.
