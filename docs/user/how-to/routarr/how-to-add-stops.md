# How to add stops

## Audience
Dispatchers and route planners.

## Purpose
Add pickup or delivery stops for a route or trip.

## Before You Start
- RoutArr access.
- Route or trip record.
- Stop address or location details.

## Steps
1. Open RoutArr.
2. Open **Routes**, **Trips**, or **Stops**.
3. Select the route or trip.
4. Open the stop list or stop creation area.
5. Enter stop details shown by the form.
6. Set stop sequence if the page allows it.
7. Save the stop.
8. Review the route or trip sequence.

## What Happens Next
RoutArr records the stop and includes it in route or trip execution.

## Troubleshooting
- If stop creation is unavailable, check route or trip status.
- If customer location is needed, customer master data is not owned by RoutArr.
- If a warehouse dock is needed, check StaffArr and LoadArr location setup.
