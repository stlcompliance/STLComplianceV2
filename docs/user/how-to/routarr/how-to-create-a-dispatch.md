# How to create a dispatch

## Audience
Dispatchers and RoutArr managers.

## Purpose
Create or prepare dispatch work through RoutArr trips, routes, or dispatch plans.

## Before You Start
- RoutArr access.
- Trip or dispatch create access.
- Origin, destination, timing, driver or equipment needs, and stops if known.

## Steps
1. Open RoutArr.
2. Open **Dispatch board** or **Dispatch plans**.
3. Review existing dispatch work.
4. Open **Trips** and select **Create** when creating the trip record.
5. Enter trip details shown by the form.
6. Add route or stop information where available.
7. Save the trip or dispatch plan.
8. Assign driver and equipment when ready.

## What Happens Next
RoutArr owns the trip, dispatch plan, status, assignments, and execution history.

## Troubleshooting
- If create is missing, check routarr.routes.create or dispatcher access.
- If dispatch is blocked, review **Validation blockers**.
- If load readiness is missing, check LoadArr.
