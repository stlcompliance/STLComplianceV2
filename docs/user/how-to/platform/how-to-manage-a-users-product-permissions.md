# How to Manage a User's Product Permissions

## Before you start
You need StaffArr permission-administration authority and the person's required work/scope details.

## Steps
1. Open StaffArr **People** and select the person.
2. Open **Permissions** or **Assignments**.
3. Review existing effective access.
4. Add the smallest appropriate role or permission set.
5. Set the correct tenant/site/department/team/record scope.
6. Review warnings and save.
7. Confirm the effective permissions.

Every ordinary product is already available to active tenant members. This workflow controls what the user may view or do inside products.
