# How to create a PM program

## Audience
Maintenance managers and admins.

## Purpose
Create a preventive maintenance program in MaintainArr.

## Before You Start
- MaintainArr access.
- PM program create access.
- Program name, asset scope, schedule, and work expectations.

## Steps
1. Open MaintainArr.
2. Open **PM programs**.
3. Enter the program details shown by the form.
4. Choose asset scope and schedule settings.
5. Review any preview or forecast shown by the page.
6. Select **Create PM program**.
7. Activate, pause, or retire the program only when the status controls allow it.

## What Happens Next
MaintainArr owns the PM program. PM generation can create work orders when automation is configured and active.

## Troubleshooting
- If **Create PM program** is missing, check MaintainArr manager or admin access.
- If PM work orders are not generated, check program status, schedule, and automation settings.
- If assets are missing from scope, check Asset registry.
