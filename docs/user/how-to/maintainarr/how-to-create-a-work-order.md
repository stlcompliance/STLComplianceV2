# How to create a work order

## Audience
Maintenance managers and authorized technicians.

## Purpose
Create a MaintainArr work order for maintenance execution.

## Before You Start
- MaintainArr access.
- Work order create access.
- Asset, priority, title, and work description.
- Assigned technician if known.

## Steps
1. Open MaintainArr.
2. Open **Work orders** or select **Create work order** from Maintenance readiness.
3. Enter the work order title and description.
4. Select the related asset when required.
5. Choose priority and planned timing fields shown by the form.
6. Assign a technician if the field is available.
7. Save or create the work order.
8. Open the work order detail to add tasks, labor, evidence, or parts demand.

## What Happens Next
MaintainArr creates the work order. It owns work execution, status, labor, evidence, and closeout.

## Troubleshooting
- If the action is missing, check MaintainArr access.
- If the asset is missing, check Asset registry.
- If assignment is blocked, check technician person reference and qualification checks.
