# How to close a work order

## Audience
Maintenance managers and admins who review finished work.

## Purpose
Close a work order after required work, evidence, and review are complete.

## Before You Start
- MaintainArr access.
- Work order close access.
- Completed task, labor, evidence, and parts information as required by your process.

## Steps
1. Open MaintainArr.
2. Open **Work orders**.
3. Select the work order.
4. Review tasks, labor, evidence, parts demand, and status history.
5. Resolve blockers before closeout.
6. Update the work order status using the status control.
7. Choose a closed or final status only when the work is reviewed and ready.
8. Save the status update.

## What Happens Next
MaintainArr records the closeout status and preserves work order history. Reports and compliance evidence can use the closed record.

## Troubleshooting
- If status update is disabled, check close permission.
- If blockers remain, resolve them before closeout.
- Do not mark work complete when required evidence is still missing.
