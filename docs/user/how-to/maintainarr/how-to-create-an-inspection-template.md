# How to create an inspection template

## Audience
Maintenance managers and admins who define inspection checklists.

## Purpose
Build an inspection checklist template in MaintainArr.

## Before You Start
- MaintainArr access.
- Inspection template manage access.
- Template name and checklist items.

## Steps
1. Open MaintainArr.
2. Open **Inspection templates**.
3. Enter the template details shown by the form.
4. Add checklist prompts and item types.
5. Mark required items where needed.
6. Review the checklist order.
7. Select **Create template**.
8. Activate or use the template according to the available status controls.

## What Happens Next
The template can be used for inspection runs when active and assigned to the right assets or programs.

## Troubleshooting
- If **Create template** is missing, check inspection manage access.
- If a template does not appear for inspection, check status and asset applicability.
- Do not use free-text notes as a substitute for required checklist items.
