# How to create a defect report

## Audience
Technicians, operators, and maintenance users allowed to report defects.

## Purpose
Report a defect against an asset for triage and possible work order creation.

## Before You Start
- MaintainArr access.
- Defect create access.
- Asset if known.
- Defect title, description, and severity.

## Steps
1. Open MaintainArr.
2. Open **Defects**.
3. Enter **Defect title**.
4. Enter **Defect description**.
5. Choose **Defect severity**.
6. Select the asset when the form requires it.
7. Select **Report defect**.
8. If work is needed and the defect status allows it, select **Open work order**.

## What Happens Next
MaintainArr creates the defect. Managers can triage it, update status, upload evidence, or open a related work order.

## Troubleshooting
- If **Report defect** is disabled, fill required fields.
- If **Open work order** is missing, the defect status may not allow it or you may lack work order create access.
- If the asset is missing, check Asset registry.
