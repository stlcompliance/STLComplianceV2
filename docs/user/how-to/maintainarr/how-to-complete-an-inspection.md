# How to complete an inspection

## Audience
Technicians and field users assigned to inspection work.

## Purpose
Complete a MaintainArr inspection run from the product workspace or Field Companion.

## Before You Start
- MaintainArr or Field Companion access.
- An inspection run in progress.
- All required checklist answers.

## Steps
1. Open MaintainArr **Inspection runs** or Field Companion **Field inbox**.
2. Select the inspection task.
3. Review the template name, asset tag, and asset name.
4. Answer each required checklist item.
5. Use Pass, Fail, N/A, number, or text fields as shown by the checklist.
6. Select **Save answers** if you need to save progress.
7. Select **Complete inspection** when all required answers are present.

## What Happens Next
MaintainArr records the inspection answers and completion result. Failed items may support follow-up defect or work workflows where configured.

## Troubleshooting
- If **Complete inspection** is disabled, required checklist items are missing or the run is not in progress.
- If the task is blocked, read the blocked reason.
- If Field Companion is offline, verify the offline queue sync status.
