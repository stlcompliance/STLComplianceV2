# How to review governing bodies

## Audience
Compliance users reviewing regulatory sources.

## Purpose
Find and review governing body catalog entries in Compliance Core.

## Before You Start
- Compliance Core access.
- Vocabulary or regulatory read access.

## Steps
1. Open Compliance Core.
2. Open **Registry**.
3. Open **Governing bodies**.
4. Search or filter the list if controls are available.
5. Select a governing body.
6. Review related jurisdictions, citations, rulepacks, or requirements.
7. Open related mappings when you need requirement context.

## What Happens Next
Users can use governing body context to understand citations, requirements, and rulepacks.

## Troubleshooting
- If governing bodies are missing, check import status and registry setup.
- If related citations are missing, review citation mappings.
- If edit actions are hidden, check compliance admin access.
