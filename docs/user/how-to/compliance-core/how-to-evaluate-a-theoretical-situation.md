# How to evaluate a theoretical situation

## Audience
Compliance admins and reviewers.

## Purpose
Evaluate a proposed or hypothetical situation against Compliance Core rules.

## Before You Start
- Compliance Core access.
- Simulation create or evaluate access where required.
- Situation facts such as subject, document type, driver, vehicle, training, or shipment context.

## Steps
1. Open Compliance Core.
2. Open **Theoretical situation**.
3. Create or select a situation.
4. Enter the facts requested by the form.
5. Select the evaluation action.
6. Review rule matches, missing facts, evidence needs, and findings.
7. Save or share the result according to your workflow.

## What Happens Next
Compliance Core records the evaluation result for review. It does not execute operational work or provide legal advice.

## Troubleshooting
- If no rule matches, check facts, rulepacks, and mappings.
- If evaluation is disabled, check compliancecore.simulation.evaluate access.
- If the situation involves real work, update the owning product record separately.
