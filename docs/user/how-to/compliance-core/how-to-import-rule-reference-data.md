# How to import rule reference data

## Audience
Compliance admins and authorized import users.

## Purpose
Bring rule, citation, or reference data into Compliance Core for review.

## Before You Start
- Compliance Core access.
- Import create access such as compliancecore.import.create or compliance admin authority.
- A valid import file or source accepted by the Imports page.

## Steps
1. Open Compliance Core.
2. Open **Imports**.
3. Choose the import action shown by the page.
4. Select or upload the source data.
5. Review import summary and validation results.
6. Fix validation errors before mapping or commit.
7. Continue to staged row review when the import is accepted.

## What Happens Next
Compliance Core creates an import batch or staged import rows. The data is not final until reviewed, mapped, and committed by an authorized user.

## Troubleshooting
- If import is blocked, check compliance admin access.
- If validation fails, correct the source file or mapping.
- If duplicate or conflicting rows appear, review staged rows before commit.
