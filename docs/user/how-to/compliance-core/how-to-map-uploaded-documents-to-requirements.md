# How to map uploaded documents to requirements

## Audience
Compliance users and records users working with evidence.

## Purpose
Connect evidence or documents to Compliance Core requirements.

## Before You Start
- Compliance Core access.
- Evidence mapping access.
- A document or record stored through RecordArr or an owning product.
- A requirement or evidence requirement in Compliance Core.

## Steps
1. Open Compliance Core.
2. Open **Evidence mapping** or **Mappings**.
3. Find the requirement or evidence requirement.
4. Select the document or evidence reference using the available picker.
5. Confirm evidence type and requirement mapping.
6. Save or confirm the mapping.
7. Review findings or missing evidence warnings after mapping.

## What Happens Next
Compliance Core understands the evidence relationship. RecordArr remains the owner of stored document files.

## Troubleshooting
- If the document is missing, check RecordArr.
- If the requirement is missing, check Mappings or Registry.
- If evidence still appears missing, refresh the compliance report or verify effective dates.
