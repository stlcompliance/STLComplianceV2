# How to review citations

## Audience
Compliance users and auditors reviewing rule support.

## Purpose
Find citations and understand related requirements or evidence.

## Before You Start
- Compliance Core access.
- Regulatory read access.
- Known citation, governing body, or requirement.

## Steps
1. Open Compliance Core.
2. Open **Mappings** > **Citations**.
3. Search or filter for the citation.
4. Select the citation.
5. Review linked requirements, rulepacks, and evidence requirements.
6. Open related findings or reports if you need current compliance posture.

## What Happens Next
Citation review helps users understand why a requirement or evidence need exists.

## Troubleshooting
- If citation is missing, check import and mapping status.
- If requirement links are missing, review requirement mapping.
- If evidence is missing, use Evidence mapping and RecordArr.
