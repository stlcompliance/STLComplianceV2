# How to view assigned work

## Audience
Field Companion users.

## Purpose
Use Field Companion to see assigned cross-product work in one mobile inbox.

## Before You Start
- You are signed in.
- Field Companion access.
- Product entitlements for source products.

## Steps
1. Open Field Companion.
2. Open **Field inbox**.
3. Review summary cards for Assigned, Blocked, Products, and Synced.
4. Use product filter chips to narrow tasks.
5. Open a task card to review title, source product, task type, status, priority, due time, and blocked reason.
6. Select **Open in [Product]** when you need the owning product workflow.

## What Happens Next
You can act from Field Companion only where the task type supports it. Final records are owned by source products.

## Troubleshooting
- If no tasks appear, clear filters or check assignments in the source product.
- If some product inboxes failed to load, open the source product or contact an admin.
- If a task is blocked, resolve the blocked reason before completing it.
