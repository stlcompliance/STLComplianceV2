# How to complete a mobile inspection

## Audience
Technicians and field workers assigned inspection tasks.

## Purpose
Complete a MaintainArr inspection from Field Companion.

## Before You Start
- Field Companion access.
- A MaintainArr inspection task in Field inbox.
- Required checklist answers.

## Steps
1. Open Field Companion.
2. Open **Field inbox**.
3. Select the MaintainArr inspection task.
4. Review **Complete inspection**.
5. Answer each checklist prompt.
6. Use **Save answers** to save progress.
7. Select **Complete inspection** when all required answers are present.
8. Watch for the success message confirming the inspection completed in MaintainArr.

## What Happens Next
Field Companion submits the answers to MaintainArr. MaintainArr owns the inspection run and result.

## Troubleshooting
- If **Complete inspection** is disabled, required answers are missing or the inspection is not in progress.
- If validation denies the task, read the reason and open MaintainArr if needed.
- If offline, check **Offline queue** and sync when connection returns.
