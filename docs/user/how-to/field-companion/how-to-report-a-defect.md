# How to report a defect

## Audience
Field workers and technicians.

## Purpose
Report a defect from the field when the owning product supports it.

## Before You Start
- Field Companion or MaintainArr access.
- A related asset if known.
- Defect title, description, and severity.

## Steps
1. Open Field Companion and check whether a defect task or product link is present.
2. If Field Companion does not show a defect form, select **Open in MaintainArr** or open MaintainArr directly.
3. Open **Defects**.
4. Enter defect title, description, and severity.
5. Select **Report defect**.
6. Attach evidence if the product workflow allows it.

## What Happens Next
MaintainArr owns the defect. Field Companion is only the field surface when available.

## Troubleshooting
- If no defect form appears in Field Companion, use MaintainArr.
- If the asset is missing, check MaintainArr Asset registry.
- If upload fails, check connection and file size.
