# How to acknowledge a task

## Audience
Field Companion users.

## Purpose
Mark that you have seen a field task without completing the underlying product workflow.

## Before You Start
- Field Companion access.
- A task in Field inbox.

## Steps
1. Open Field Companion.
2. Open **Field inbox**.
3. Find the task.
4. Review task title, status, priority, and blocked reason.
5. Select **Acknowledge**.
6. Confirm the button changes to **Acknowledged**.
7. Open the owning product if more work is required.

## What Happens Next
Field Companion records local or synced acknowledgement. The owning product task may still need completion.

## Troubleshooting
- If **Acknowledge** is disabled, it may already be acknowledged.
- If acknowledgement does not sync, check Offline queue.
- Do not treat acknowledgement as completion unless the task explicitly says so.
