# How to complete a training step from mobile

## Audience
Trainees using Field Companion for an assigned TrainArr step.

## Purpose
Work on training from a mobile surface when supported.

## Before You Start
- TrainArr assignment.
- Field Companion access or TrainArr mobile access.
- Required training evidence or signoff access.

## Steps
1. Open Field Companion.
2. Open **Field inbox**.
3. Select the TrainArr task if present.
4. Upload evidence if the task shows an evidence panel.
5. Use **Open in TrainArr** for evaluation or signoff actions.
6. In TrainArr, complete evaluation or signoff steps shown by the assignment.
7. Confirm the assignment status updates.

## What Happens Next
TrainArr owns training completion, evaluation, signoffs, and qualifications.

## Troubleshooting
- If no TrainArr task appears, check assignment, permission, and synchronization.
- If signoff is missing, check whether you are allowed to sign as trainee or trainer.
- If completion is blocked, check missing evaluation or signoffs.
