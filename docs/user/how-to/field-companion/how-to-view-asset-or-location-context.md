# How to view asset or location context

## Audience
Field workers, inspectors, drivers, technicians, and supervisors.

## Purpose
Use a field task to understand the asset, location, dock, or work context before acting.

## Before You Start
- Field Companion access.
- An assigned task.
- Device connectivity or offline task data.

## Steps
1. Open Field Companion.
2. Open the field inbox.
3. Select the task.
4. Review the task header and detail panels for asset, location, source product, source record, due time, and instructions.
5. Use scan or task context controls if the tenant provides them.
6. Open the related action panel for inspection, work order, receiving, or evidence when needed.
7. Do not edit source context from Field Companion unless the task explicitly provides an owning-product action.
8. Escalate incorrect asset or location data to the owning product.

## What Happens Next
Field Companion helps field users act with context while source records remain in their owning products.

## Troubleshooting
- If asset context is wrong, correct it in MaintainArr.
- If location context is wrong, correct it in StaffArr or the owning workflow that selected the location.
- If a task is missing context while offline, reconnect and refresh the task before acting.
