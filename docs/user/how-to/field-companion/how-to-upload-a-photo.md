# How to upload a photo

## Audience
Field users who need to attach evidence.

## Purpose
Upload evidence through a supported Field Companion task.

## Before You Start
- Field Companion access.
- A task that supports evidence upload.
- A photo or file available on the device.

## Steps
1. Open Field Companion.
2. Open **Field inbox**.
3. Select a task that shows an evidence upload panel.
4. Choose the file or photo using the upload control.
5. Enter evidence notes or evidence type if shown.
6. Submit the upload.
7. Wait for the synced or success status.

## What Happens Next
The evidence is submitted to the owning product or RecordArr-controlled record flow depending on task type.

## Troubleshooting
- If no upload control appears, the task type may not support direct evidence upload.
- If upload fails, check connection and retry.
- If the task is offline, check **Offline queue**.
