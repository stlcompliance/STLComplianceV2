# How to update vendor order status

## Audience
SupplyArr users or vendor users with vendor order access.

## Purpose
Update or review vendor order progress in SupplyArr.

## Before You Start
- A vendor order.
- SupplyArr access or a vendor portal link.
- Permission or invitation to update status.

## Steps
1. Open SupplyArr.
2. Open **Purchasing** > **Vendor orders**.
3. Select the vendor order.
4. Review the current status and line details.
5. Use the visible status update or document upload action if available.
6. For vendors, open the vendor portal link and follow the portal prompts.
7. Save or submit the update.

## What Happens Next
SupplyArr records the vendor order status. LoadArr uses receiving or expected receipt workflows for physical goods.

## Troubleshooting
- If status update is unavailable, your role or portal invitation may be read-only.
- If documents are required, upload them through the vendor order document action where available.
- If goods arrived, complete receiving in LoadArr.
