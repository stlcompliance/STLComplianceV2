# How to create a part

## Audience
SupplyArr admins, managers, or buyers who manage item and part catalog data.

## Purpose
Create a purchasable item or part in SupplyArr.

## Before You Start
- SupplyArr access.
- Part management access.
- Part key, display name, unit of measure, vendor context if needed.

## Steps
1. Open SupplyArr.
2. Open **Catalog**.
3. Select the create or add part action shown by the page.
4. Enter part key and display name.
5. Enter unit of measure and other required catalog details.
6. Connect vendor or price context where available.
7. Save the part.
8. Confirm it appears in the Catalog and purchasing pickers.

## What Happens Next
SupplyArr owns the commercial part or item context. LoadArr owns inventory balances and stock movement for physical stock.

## Troubleshooting
- If the part is missing from a purchase request picker, confirm it is saved and active.
- If inventory quantity is wrong, check LoadArr instead of editing the part.
- If vendor price is wrong, review pricing snapshots where enabled.
