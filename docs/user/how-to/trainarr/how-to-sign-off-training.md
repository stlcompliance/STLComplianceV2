# How to sign off training

## Audience
Trainees and trainers who are allowed to sign an assignment.

## Purpose
Submit trainee or trainer signoff for a TrainArr assignment.

## Before You Start
- TrainArr access.
- An open training assignment.
- A passing evaluation if completion depends on evaluation.
- Signoff access for the required role. Current implemented signoff roles are trainee and trainer.

## Steps
1. Open TrainArr.
2. Open **Assignments**.
3. Select the assignment.
4. Review **Evaluation & signoffs**.
5. Enter Signoff notes if needed.
6. If you are the trainee and the button is shown, select **Trainee signoff**.
7. If you are the trainer and the button is shown, select **Trainer signoff**.
8. Review the assignment status after signing.

## What Happens Next
TrainArr records the signoff. Completion requires a passing evaluation plus trainee and trainer signoffs.

## Troubleshooting
- If the signoff button is missing, your role may not allow that signoff or it may already be signed.
- If completion remains blocked, check evaluation result and missing signoffs.
- If the assignment is closed or expired, signoff may no longer be available.
