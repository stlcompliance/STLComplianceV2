# How to issue a certificate

## Audience
Training admins and trainers who manage qualification completion.

## Purpose
Understand how TrainArr moves from completed assignment to qualification or certification status.

## Before You Start
- TrainArr access.
- A training assignment.
- Passing evaluation.
- Trainee signoff and trainer signoff.

## Steps
1. Open TrainArr.
2. Open **Assignments**.
3. Select the assignment.
4. Confirm the evaluation result is **pass**.
5. Confirm the trainee and trainer signoffs are present.
6. Use the completion action when it is available for the assignment.
7. Open **Qualifications** to review the resulting qualification issue or status.
8. Check StaffArr **Certifications** for published snapshots where publication is enabled.

## What Happens Next
TrainArr owns the qualification or certification state and may publish the result to StaffArr for readiness display.

## Troubleshooting
- If no certificate appears, check missing requirements.
- If StaffArr does not show the result, check publication settings or sync.
- If the certificate should be suspended, revoked, or expired, use TrainArr qualification controls where available.
