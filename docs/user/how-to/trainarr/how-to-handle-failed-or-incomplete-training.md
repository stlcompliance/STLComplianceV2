# How to handle failed or incomplete training

## Audience
Training admins, trainers, managers, and qualification reviewers.

## Purpose
Record failed or incomplete training and decide the next remediation step.

## Before You Start
- TrainArr access.
- Assignment view or evaluation access.
- Training assignment, step evidence, signoff status, and evaluation details.

## Steps
1. Open TrainArr.
2. Open **Assignments**.
3. Select the assignment.
4. Review steps, evidence, signoffs, and evaluation status.
5. Submit the practical evaluation result when required.
6. If the trainee did not pass or did not complete requirements, keep the assignment incomplete or record the failed result according to the evaluation form.
7. Create or review remediation in the remediation area if follow-up training is needed.
8. Notify the manager or StaffArr owner if work readiness is affected.

## What Happens Next
TrainArr keeps the training state and can publish blockers or qualification changes for StaffArr and downstream products.

## Troubleshooting
- If the assignment cannot be evaluated, verify evaluator permission.
- If a failed result should restrict work, coordinate with StaffArr readiness or restrictions rather than editing another product.
- If remediation does not appear, create a new assignment or follow the tenant's manual remediation process.
