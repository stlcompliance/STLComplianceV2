# How to review expiring certifications

## Audience
Trainers, managers, and StaffArr or compliance users.

## Purpose
Find qualifications or certifications that need attention before they expire.

## Before You Start
- TrainArr or StaffArr access.
- Permission to view qualification or certification status.

## Steps
1. Open TrainArr.
2. Open **Qualifications**.
3. Review qualification issues and status history.
4. Look for statuses such as issued, suspended, expired, revoked, or expiring warnings where shown.
5. Open StaffArr **Certifications** to see published qualification or certification snapshots for people.
6. Create remediation or retraining assignments when required and available.

## What Happens Next
Users can plan renewal or retraining before readiness is blocked.

## Troubleshooting
- If certifications are missing in StaffArr, check TrainArr publication.
- If a person is missing, check StaffArr person records.
- If expiration dates are wrong, review the TrainArr qualification rule or program setup.
