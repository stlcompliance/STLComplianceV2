# How to assign retraining after an incident

## Audience
StaffArr admins, trainers, and managers handling training-compliance incidents.

## Purpose
Route a personnel incident toward TrainArr retraining when the incident category supports it.

## Before You Start
- A StaffArr personnel incident.
- Reason category **Training compliance** if routing is required.
- TrainArr access and assignment permissions.
- A retraining program or definition.

## Steps
1. Open StaffArr.
2. Open the person or **Incidents**.
3. Select the incident.
4. Confirm the reason category is **Training compliance**.
5. Use the TrainArr routing action when it is visible.
6. Open TrainArr **Assignments** or **Remediation**.
7. Create or review the retraining assignment.
8. Track completion, evaluation, and signoffs in TrainArr.

## What Happens Next
StaffArr keeps the incident history. TrainArr owns retraining assignment and completion.

## Troubleshooting
- If routing is not visible, check incident category and permissions.
- If no retraining program is available, create or activate a program first.
- If completion does not update readiness, check TrainArr qualification publication to StaffArr.
