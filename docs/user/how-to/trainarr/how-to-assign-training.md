# How to assign training

## Audience
Training admins, trainers, or managers allowed to create assignments.

## Purpose
Assign a training definition to a StaffArr person.

## Before You Start
- TrainArr access.
- Permission to manage assignments.
- A StaffArr person record.
- An available training definition or program.

![TrainArr manual assignment screen showing person, training definition, due date, and assignment status.](/screenshots/trainarr-assignment.png "Create the assignment, then review evidence, evaluation, and signoff status.")

## Steps
1. Open TrainArr.
2. Open **Assignments**.
3. Choose **Manual** if assigning directly.
4. Select the StaffArr person.
5. Select the training definition.
6. Review any qualification check or applicability result shown.
7. Create the assignment.
8. Open the assignment detail to review evidence, evaluation, and signoff status.

## What Happens Next
TrainArr creates a training assignment. The trainee and trainer can complete evidence, evaluation, and signoffs as permitted.

## Troubleshooting
- If the person is missing, check StaffArr.
- If the training definition is missing, check Programs.
- If assignment creation is blocked by applicability or qualification checks, resolve the blocker before assigning.
