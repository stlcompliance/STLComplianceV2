# How to create a training program

## Audience
Trainers and training admins.

## Purpose
Create a TrainArr training program and define its training structure.

## Before You Start
- TrainArr access.
- Permission to manage programs.
- Program name, target audience, module or step content, and completion expectations.

## Steps
1. Open TrainArr.
2. Open **Programs** and select **Create**.
3. Enter the program details shown by the form.
4. Add training steps with **Add step** where the step builder is available.
5. Add branch rules with **Add branch rule** only when a branch is needed.
6. Define pass criteria and any required signoff settings shown by the form.
7. Save the program.
8. Review the program in **Programs** > **Details**.

## What Happens Next
The program becomes available for training assignments when active and allowed by the training setup.

## Troubleshooting
- If **Create** is missing, check training admin access.
- If a step cannot be saved, complete all required fields shown by the form.
- If assignments cannot use the program, confirm the program is active and has required content.
