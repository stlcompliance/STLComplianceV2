# How to create a location

## Audience
StaffArr admins who manage internal places.

## Purpose
Create an internal operational location such as a building, room, dock, yard, or area.

## Before You Start
- StaffArr access.
- Permission to manage locations.
- An active site if the location belongs under a site.

## Steps
1. Open StaffArr.
2. Open **Locations** or **Organization structure** with the locations tab selected.
3. Select the action to add a location.
4. Choose the site or parent location.
5. Enter the location name.
6. Enter any required address or path details shown by the form.
7. Save the location.
8. Confirm the location appears in the location list or picker.

## What Happens Next
The location becomes the StaffArr-owned internal place reference. Other products may reference it but should not own a duplicate location truth.

## Troubleshooting
- If the site is missing, create the site first.
- If the location is missing in another product, check that the product sync or reference list is current.
- If the location should not be used yet, keep it inactive or planned where supported.
