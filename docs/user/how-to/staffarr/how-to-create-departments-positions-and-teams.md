# How to create departments, positions, and teams

## Audience
StaffArr admins who maintain workforce structure.

## Purpose
Create the org units needed for person placement and scoped role assignments.

## Before You Start
- StaffArr access.
- Permission to manage org structure.
- Names and parent relationships for departments, positions, and teams.

## Steps
1. Open StaffArr.
2. Open **Org structure**.
3. Create each Department with unit type **Department**.
4. Create each Team with unit type **Team** and select a team type such as Operational, Maintenance, Warehouse, Dispatch, Safety, Quality, Training, Admin, Project, or Emergency response.
5. Create each Position with unit type **Position**.
6. Set each unit status to **Active** when it is ready for assignment.
7. Save each org unit.
8. Use the units in person placement or role scopes.

## What Happens Next
Active departments, teams, and positions become available in person creation, role scope, and reporting context.

## Troubleshooting
- If a unit does not show in person creation, confirm its type and active status.
- If the structure is wrong, edit the parent relationship before assigning many people.
- Avoid using free-text job titles as a substitute for canonical positions when position org units are needed.
