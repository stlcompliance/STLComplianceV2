# How to create an organization unit

## Audience
StaffArr admins who maintain org structure.

## Purpose
Create a canonical internal org unit in StaffArr.

## Before You Start
- StaffArr access.
- Permission to manage org structure.
- Name, unit type, parent unit if applicable, and status.

## Steps
1. Open StaffArr.
2. Open **Org structure** or **Organization structure**.
3. Choose the organization tab if tabs are shown.
4. Select the action to add or create an org unit.
5. Enter the org unit name.
6. Choose the unit type, such as Company, Division, Region, Business unit, Cost center, Site, Department, Team, Position, or Other.
7. Choose status such as Planned, Active, Inactive, or Archived.
8. Choose the parent unit if the org unit belongs under another unit.
9. Save the org unit.

## What Happens Next
The org unit becomes available for StaffArr placement, role scope, and product reference where active.

## Troubleshooting
- If the unit type you need is not shown, use the closest existing option or ask an admin before creating a workaround.
- Inactive or archived units may not appear in normal placement pickers.
- Products should consume StaffArr org units instead of maintaining their own internal org hierarchy.
