# PM Program to Work Order

## Purpose
Use a preventive maintenance program to drive future maintenance work.

## Who Participates
- Maintenance manager
- Technician

## Starting Point
A PM program is needed for an asset or asset group.

## Main Steps
1. Create the PM program in MaintainArr.
2. Set asset scope and schedule.
3. Activate the program where allowed.
4. Review PM due or forecast results.
5. Generate or create work orders.
6. Execute and close work orders.

## Products Involved
- MaintainArr owns PM programs and work orders.
- StaffArr owns technician/person context.
- RecordArr may store evidence.

## Records Created or Updated
- PM program
- PM schedule or occurrence
- work order
- inspection run
- maintenance history

## Where Users May Get Stuck
- Program not active.
- Automation settings missing.
- Asset scope does not include expected assets.
- Work order assignment blocked.
