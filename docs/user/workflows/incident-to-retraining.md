# Incident to Retraining

## Purpose
Route a personnel incident toward retraining when training compliance is involved.

## Who Participates
- StaffArr admin or HR user
- Trainer
- Manager
- Trainee

## Starting Point
A personnel incident is recorded in StaffArr.

## Main Steps
1. Report the incident in StaffArr.
2. Set reason category to Training compliance when applicable.
3. Review severity and notes.
4. Use TrainArr routing where visible.
5. Create or review the retraining assignment in TrainArr.
6. Complete evaluation and signoffs.
7. Review StaffArr readiness or certification updates.

## Products Involved
- StaffArr owns personnel incident history.
- TrainArr owns retraining.
- RecordArr may store evidence.
- ReportArr may report trends.

## Records Created or Updated
- personnel incident
- incident note or attachment
- training assignment
- evaluation
- signoff

## Where Users May Get Stuck
- Routing action missing because category or permission is wrong.
- No retraining program exists.
- Qualification update not published to StaffArr.
