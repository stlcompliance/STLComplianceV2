# Dispatch to Completion

## Purpose
Plan, execute, and close transportation work in RoutArr.

## Who Participates
- Dispatcher
- Driver
- Warehouse user
- Maintenance manager when readiness blocks appear

## Starting Point
A trip or dispatch plan is needed.

## Main Steps
1. Create trip or dispatch work in RoutArr.
2. Create or select route and stops.
3. Assign driver and equipment.
4. Review validation blockers.
5. Update trip status during execution.
6. Submit proof or DVIR where required.
7. Review proof and complete trip.

## Products Involved
- RoutArr owns trip, route, dispatch, and proof.
- StaffArr owns driver person record.
- TrainArr owns driver qualification.
- MaintainArr owns vehicle readiness.
- LoadArr owns load readiness.

## Records Created or Updated
- trip
- route
- stop
- driver assignment
- vehicle assignment
- proof record
- exception

## Where Users May Get Stuck
- Driver or vehicle assignment blocked.
- Load readiness missing.
- Proof missing or rejected.
- Missed stop unresolved.
